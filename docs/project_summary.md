# Project Summary

## Project Title

St. John's Parking Analytics Pipeline

## Summary

This project analyzes parking activity in St. John's using CSV datasets, SQL, and Power BI. The solution cleans raw parking data, combines it with weather information, and creates analysis outputs that help understand parking demand, revenue, payment behaviour, and winter weather impact.

## Skills Demonstrated

- CSV data handling
- SQL querying
- ETL process
- Data cleaning
- Data mapping
- Business requirements documentation
- UAT test case writing
- Power BI dashboard planning
- Business insight generation
